WEB-технологии

Основы WEB-сайта

HTML

HTML (HyperText Markup Language, Язык Гипертекстовой Разметки) — это стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере. Он является основой WEB-страницы. HTML-код представляет из себя множество упорядоченных контейнеров, которые изнутри состоят из простых элементов. Язык HTML предоставляет средства для создания заголовков, абзацев, списков, ссылок, цитат и других элементов. Элементы HTML выделяются тегами, записанными с использованием угловых скобок. Браузеры не отображают HTML-теги, но используют их для интерпретации содержимого страницы. В языке регулярно появляются новые теги и возможности. За внедрение новых рекомендаций отвечает W3C.

CSS

CSS (Cascading Style Sheets, Каскадные Таблицы Стилей) — это язык описания внешнего вида документа (web-страницы), написанного с использованием языка разметки, например HTML или XHTML. Также он может применяться к любым XML-документам, например, к SVG или XUL. CSS описывает внешний вид как отдельных элементов, так и всего документа в целом. CSS используется для задания цветов, шрифтов, стилей, расположения отдельных блоков и других аспектов представления внешнего вида веб-страниц. Основная цель CSS — ограждение и отделение описания логической структуры веб-страницы от описания её внешнего вида. За добавления новых функций в язык также отвечает W3C.

JavaScript

JavaScript — это мультипарадигменный язык программирования, который поддерживает объектно-ориентированный, императивный и функциональный стили программирования. Он является реализацией спецификации ECMAScript. JavaScript отвечает за функцианальную часть сайта.

Объектные модели

BOM (Browser Object Model, «Объектная Модель Браузера») — это соглашение для конкретного браузера, относящееся ко всем объектам, предоставляемым веб-браузером. В отличие от объектной модели документа, здесь нет ни стандарта для реализации, ни строгого определения, поэтому поставщики браузеров могут свободно реализовывать спецификацию по своему усмотрению.

То, что мы видим как окно, отображающее документ, программа-браузер воспринимает как иерархический набор объектов. Когда браузер анализирует документ, он создает набор объектов, определяющих документ и детализирующих способ его отображения. Объект, который создает браузер, известен как объектная модель документа (DOM). Это часть большой коллекции объектов, которую использует браузер. Эта коллекция объектов браузера известна под общим названием объектной модели браузера или BOM.

DOM (Document Object Model, «Объектная Модель Документа») — это независящий от платформы и языка программный интерфейс, позволяющий программам и скриптам получить доступ к содержимому HTML-, XHTML- и XML-документов, а также изменять содержимое, структуру и оформление таких документов.

Модель DOM не накладывает ограничений на структуру документа. Любой документ известной структуры с помощью DOM может быть представлен в виде дерева узлов, каждый узел которого представляет собой элемент, атрибут, текстовый, графический или любой другой объект. Узлы связаны между собой отношениями «родительский-дочерний».
